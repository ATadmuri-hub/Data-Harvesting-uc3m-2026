1 Research Question and Political Context

Research Question
Does the frequency of official government immigration statements correlate with shifts in PSOE electoral support over the 2023–2026 governing period, and can historical data produce a credible 2027 electoral forecast?

Why This Topic?

Spain offers a rare natural experiment. Pedro Sánchez leads a fragile left-wing minority government (dependent on Junts, ERC, Bildu, PNV) that has faced three overlapping pressures since November 2023:

  • Record Canary Islands arrivals via the Atlantic migration route throughout 2024
  • Blocked distribution of unaccompanied minors (MENA) between Autonomous Communities in 2024–2025
  • An extraordinary regularisation decree in January 2026 targeting ~500,000 undocumented migrants — the first mass legalisation since Zapatero’s 2005 process

This makes Spain an ideal case to study whether government communication on immigration helps or hurts the governing party electorally.

Hypotheses

HypothesisDirectionResult
H1 (Framing): Government press releases on immigration reassure centrist votersPSOE improvesNot supported (r = 0.074)
H2 (Issue Ownership): Higher media salience benefits right-leaning parties (Petrocik, 1996)PSOE fallsSupported (r = −0.699)
H3 (Attrition): Mid-term governmental fatigue is a democratic regularityPSOE fallsSupported (β = −0.138 pp/month)

Study Window

28 months: November 2023 (Sánchez investiture) through February 2026 (regularisation decree aftermath). PSOE polling fell from 29.1% to 25.8% — a cumulative decline of 3.3 percentage points.

3 Source 1 — La Moncloa Press Releases

The Problem
La Moncloa’s press index page (index.aspx?Anyo=&Mes=) uses client-side JavaScript filtering. The URL parameters are ignored by the server — every HTTP request returns today’s content regardless of the year/month you specify. Standard scraping tools (rvest, httr2) see only the current page, not historical press releases.

The Solution: Two-Stage Architecture

Stage 1 — URL Discovery via Wayback Machine CDX API. The Internet Archive’s CDX API indexes historical snapshots of web pages. We queried it for all archived URLs from three immigration-related ministries (Interior, Inclusión, Exteriores) between November 2023 and February 2026. This gave us 647 unique URLs after case-insensitive deduplication.

# CDX API query for each ministry/year combination
cdx_url <- paste0(
  "https://web.archive.org/cdx/search/cdx",
  "?url=www.lamoncloa.gob.es/ministerios/", ministry, "/*",
  "&output=json&limit=5000",
  "&from=", "20231101",
  "&to=",   "20260301",
  "&filter=statuscode:200"
)

Stage 2 — Live HTML Scraping with XPath. The 647 CDX URLs were filtered by immigration slug keywords (25 terms like inmigra, regulariza, asilo, frontera), yielding 64 candidate URLs. Each was fetched from the live La Moncloa server and parsed using XPath:

# XPath: find <h1> anywhere in the DOM tree (descendant axis)
title <- page |>
  xml_find_first("//h1") |>
  xml_text(trim = TRUE)

# XPath union: try multiple body containers with contains() function
text_nodes <- page |>
  xml_find_all(
    "//div[contains(@class,'noticia-texto')]//p | //div[contains(@class,'cuerpo-noticia')]//p | //article//p"
  )

# Regex: extract date from URL slug
date_match <- str_extract(slug, "^\\d{6,8}")

4 Source 2 — Wikipedia Opinion Polls

Target: The Wikipedia article “Opinion polling for the next Spanish general election” contains 5 wikitables (one per year + historical CIS series) with polling firm, dates, sample size, and vote shares per party.

The Image Header Problem
Wikipedia polling tables use party logos (<img> elements) as column headers instead of text. The standard html_table() function cannot read them — it returns empty or “unknown” column names. We solved this with XPath by extracting the alt attribute from the <img> tags inside each <th> cell. 
# XPath: extract party names from image alt attributes in header row
header_ths <- tbl |> xml_find_all(".//tr[1]/th")
col_names <- map_chr(header_ths, function(th) {
  txt <- xml_text(th, trim = TRUE)
  if (nchar(txt) > 0) return(txt)
  # Fallback: read the image alt attribute
  img_alt <- th |> xml_find_first(".//img") |> xml_attr("alt")
  if (!is.na(img_alt)) return(img_alt)
  return("unknown")
})

Percentage columns contained ranges (e.g., "28.1–29.3%") that required regex midpoint extraction. Fieldwork dates spanned two rows and needed lubridate::dmy() with fallback logic. The 47+ polling firms had inconsistent name formatting that was standardised.

5 Source 3 — INE Demographics (REST/JSON API)

The INE (Instituto Nacional de Estadística) provides a public REST/JSON API at servicios.ine.es/wstempus/js/ES/. No authentication required. We fetched two series:

1. EPA Unemployment by Province
Table 65349, Operation 293 (Encuesta de Población Activa). Quarterly data for all 47 Spanish provinces. 477 series covering unemployment rates by sex.
2. Foreign Population Trend
Table 9683, filtered to total non-Spanish population at national level. Variables: tv=70032:16420 (total nationality), tv=70013:451 (ambos sexos). Foreign = total − Spanish. 
# httr2 request builder with retry logic and timeout
resp <- request(url) |>
  req_headers(`User-Agent` = UA) |>
  req_retry(max_tries = 3, backoff = ~10) |>
  req_timeout(30) |>
  req_perform()

# Parse JSON response
data <- resp |> resp_body_json()

The INE API uses a custom FK_Periodo encoding for time periods (e.g., code 22 = Q4 2025, code 21 = Q3 2025). We wrote a mapping function to convert these codes to standard quarter dates. The API also has volume restrictions — large tables sometimes return error JSON instead of data, which our code handles gracefully with fallback placeholder CSVs.

6 Source 4 — El País Media Salience

Target: El País immigration topic section (elpais.com/noticias/inmigracion/) and general Spain section (elpais.com/espana/).

The Search Problem
El País does not offer a public search API. The site’s internal search returns 404 for programmatic requests. However, the topic tag pages work with static pagination: /noticias/inmigracion/1/, /noticias/inmigracion/2/, etc. Each page lists ~40–45 articles with headlines and ISO 8601 timestamps.

We paginated through the immigration topic section (83 pages) and the general Spain section (40 pages), extracting headlines and dates using XPath:

# XPath union: extract all h2 and h3 elements (headline containers)
headlines <- page |>
  xml_find_all("//h2 | //h3") |>
  xml_text(trim = TRUE)

# XPath attribute selector: find <time> elements with datetime attribute
times <- page |>
  xml_find_all("//time[@datetime]") |>
  xml_attr("datetime")

The salience index is constructed as: n_immigration / max(n_immigration) × 100, normalised to 0–100 where the peak month scores 100. This captures media agenda intensity, not just volume.

8 Statistical Analysis and Modeling

Script 06 (667 lines) is the analytical core. It builds the 28-month master panel by merging all four data sources on year_month, then runs a full suite of time-series models.

Master Panel Construction

The panel has 28 rows × 18 columns including: PSOE/PP/Vox/Sumar averages, number of polls per month, government statement count, El País immigration article count, normalised salience index, national unemployment rate, time trend variable, and lagged versions of key predictors.

Models Estimated

All standard errors are Newey-West HAC (heteroscedasticity and autocorrelation consistent) to account for serial correlation in monthly time series. Additionally:

  • Granger causality tests — Government statements do not Granger-cause PSOE polling (F = 0.147, p = 0.864)
  • ARIMA(0,1,0)-X model — Selected via AIC/BIC, with media salience as exogenous regressor
  • ETS model — Exponential smoothing for comparison
  • VAR(1) bivariate — Joint PSOE + PP forecast (exploiting their negative correlation, r = −0.60)
  • Rolling one-step-ahead forecast validation — ARIMA vs ETS accuracy comparison
  • Full diagnostic suite — ADF test, Phillips-Perron test, Ljung-Box test on residuals

Three Forecast Scenarios to June 2027

The ARIMA-X model projects PSOE voting intention under three scenarios differentiated by future media salience levels:

9 D'Hondt Seat Projections (Monte Carlo)

Script 07 converts vote-share forecasts into Congress seat counts using the actual Spanish electoral system: 52 constituencies (50 provinces + Ceuta + Melilla), 350 total seats, D'Hondt divisor method with a 3% threshold per constituency.

The script includes a calibration check against the actual 2023 election results (all parties within ±4 seats of actual), then runs 5,000 Monte Carlo simulations per scenario to produce probabilistic seat distributions.

10 Historical Comparison — 2005 Zapatero Regularisation

Script 08 scrapes 2004–2008 CIS monthly polls from Wikipedia (with embedded fallback data if scraping fails) and runs an Interrupted Time Series (ITS) analysis centred on the February 2005 regularisation.

The ITS model with Newey-West robust SEs found that after the 2005 regularisation, PSOE’s declining trend reversed: the party recovered at +0.493 pp/month, eventually winning the 2008 election at 43.9%. However, Zapatero governed in a strong economic environment (GDP growth >3%), while Sánchez faces a fragmented parliament and elevated cost-of-living concerns. Direct comparison is limited.

12 Challenges Encountered and Solutions

ChallengeWhat Went WrongSolution Applied
La Moncloa JavaScript filtering The index page renders content via client-side JS. HTTP GET requests to index.aspx?Anyo=2024&Mes=6 return today’s content, not June 2024. Used the Wayback Machine CDX API for URL discovery, then fetched individual press release pages directly from the live server.
Wikipedia image headers Party columns use <img> logos instead of text. html_table() returns empty column names. XPath extraction of img/@alt attributes from each <th> cell in the first row.
INE API volume limits Large table requests sometimes return an error JSON ("No puede realizar esta consulta") instead of data. Fetch separate series endpoints. Compute foreign = total − Spanish. Graceful fallback with placeholder CSVs when endpoints fail.
CIS data access CIS (Centro de Investigaciones Sociológicas) monthly barometers require manual PDF download. No API. Switched to Wikipedia as the polling aggregation source. Wikipedia covers 47+ polling firms including CIS.
El País search The site’s search API returns 404 for programmatic requests. No public search endpoint. Used static topic tag pages (/noticias/inmigracion/N/) which support clean pagination without authentication.
RSelenium reproducibility RSelenium requires a local browser driver (Chrome/Firefox) whose version must match the installed browser. This breaks reproducibility across machines. Avoided Selenium entirely. All scraping uses rvest + httr2 only, ensuring the pipeline runs with zero system dependencies beyond R packages.

16 Key Findings

Finding 1: PSOE Support in Persistent Decline
PSOE support fell −0.138 pp/month (OLS, p < 0.001). Cumulative decline: 3.3 pp from 29.1% (Nov 2023) to 25.8% (Feb 2026). The trend is robust across all model specifications (R² = 0.695–0.727).
Finding 2: Media Immigration Salience Suppresses PSOE Support
Pearson r = −0.699 (p < 0.001). Months with highest El País immigration coverage consistently coincide with lowest PSOE polling. Consistent with Petrocik’s (1996) issue ownership theory: when immigration becomes salient, voters associate the issue with right-leaning parties (PP, Vox), not the governing left.
Finding 3: La Moncloa Cannot Lead the Narrative
Government immigration statements have zero predictive power for PSOE support. Granger test: F = 0.147, p = 0.864. Bivariate correlation between statements and PSOE is r = 0.074 (near-zero). No CCF lag exceeds the significance threshold. La Moncloa’s communication is reactive (press release volume tracks media salience) rather than strategic.
Finding 4: The January 2026 Regularisation Adds Zero Direct Voters
All ~500,000 beneficiaries remain ineligible for national suffrage through 2027. Residency ≠ citizenship. The electoral consequence is entirely mediated via issue salience — which our model shows hurts PSOE.
Finding 5: 2027 Seat Projection
PP 136–137 seats, PSOE ~120. PP + Vox coalition reaches majority (171 seats). PSOE + Sumar + Regional falls short (179 seats in best case). The regularisation debate itself is the swing variable — not the policy outcome.
